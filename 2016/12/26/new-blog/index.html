函数巩固
函数的组成：
1.函数关键字：function
2.函数名称：自定义标识符
3.一对圆括号：其中括号的内容是0个或者多个用英文逗号分隔开的标识符组成的列表，称为形参，它们就好像在函数中用var定义的局部变量一样；
4.一对花括号：其中是有多条js语句组成的函数体，一旦函数调用，就会依次执行这些语句。

函数的调用方式：
1.作为普通函数调用：函数名称(实参列表) 或者 函数名称()
2.作为对象的属性方法: 对象名.函数名称(实参列表) 或者 函数名称()
3.作为构造对象的构造函数：new 函数名称(实参) 或 new 函数名称()

实参对象：
Arguments：这个对象使用在函数定义体，当函数实际调用执行的时候，arguments会被自己赋值，他是个类数组对象，是由实际传入的每个实参组成的列表，因此最大的用途就是可以方便操作任意数量的实参。

数字类型与字符串类型
数字与字符串类型在js中是互通的，例如3 转化为字符串就是”3”, 3 == “3”, js比较会进行自动类型转化。
两者互相转化的方法：
方法一：利用String(),toString() 与Number();
方法二：
数字转化为字符串，可以通过+一个空字符串，因为数字与字符串进行+运算，数字类型会自动转化为字符串; 例如: 4+ “” —>“4”
字符串转化为数字，可以通过-0, 因为数字与字符串进行- 运算，字符串类型会自动类型转化为数字类型；例如：”4” -0 —>4

正则表达式（day06专题）
正则表达式定义：正则表达式（英语：Regular Expression，在代码中常简写为regex、regexp或RE）使用单个字符串来描述、匹配一系列符合某个句法规则的字符串搜索模式。搜索模式可用于文本搜索和文本替换。通俗的说，用一个特殊的“字符序列”表示一系列的字符串。这样也就可以通过这个字符序列搜索符号这种形式的字符串，以及搜索后用某个字符串替换这些字符串。
格式：/pattern/modifiers
修饰符：i（ignore忽略大小写）,g(global 全局), m(mutil 多行)
正则表达式常用方法：
search() –查询位置
定义：用于检索字符串中指定的子字符串，或检索与正则表达式相匹配的子字符串，并返回子串的起始位置。
举例：
var str = “Visit bbb”;
var n = str.search(/bbb/i);
用法二：search() 方法使用字符串search 方法可使用字符串作为参数。字符串参数会转换为正则表达式：检索字符串中 “w3cschool” 的子串：var str = “Visit aaa!”;
var n = str.search(“aaa”);
replace() – 修改
定义：用于在字符串中用一些字符替换另一些字符，或替换一个与正则表达式匹配的子串。
var str = “Visit aaa!”;
var res = str.replace(/aaa/i, “bbb”); //Visit bbb!
test()– 检测匹配
定义：检测一个字符串是否匹配某个模式，如果字符串中含有匹配的文本，则返回 true，否则返回 false。
//用于搜索字符串中的字符 “e”
var patt = /e/;
patt.test(“The best things in life are free!”);字符串中含有 “e”，所以该实例输出为：true
exec() –
定义：exec() 方法用于检索字符串中的正则表达式的匹配。该函数返回一个数组，其中存放匹配的结果。如果未找到匹配，则返回值为 null。
ar /fe/.exec(“The best things in life are free!”) //[“fe”]
